# Drum Kit Project

## Descrição

Este é um projeto simples de Drum Kit desenvolvido como parte do **Curso Desenvolvedor Full Stack Turbo da Tipscode**, no módulo de front-end.

A aplicação permite que o usuário toque sons de bateria tanto clicando nos botões na tela quanto pressionando teclas do teclado.

O principal objetivo deste projeto é praticar conceitos fundamentais de JavaScript por meio da construção de uma aplicação interativa no navegador.

---

## Funcionalidades

* Reprodução de sons ao clicar nos botões
* Reprodução de sons via teclado
* Feedback visual com animação dos botões
* Áudios pré-carregados para melhor performance
* Código limpo e escalável

---

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla)

## Principais Aprendizados

### Funções Callback

Compreensão de como passar funções como argumento e utilizá-las em event listeners.

### Orientação a Objetos

Organização do código de forma mais estruturada, agrupando dados e comportamentos relacionados.

### Criação de Objetos Mais Complexos

Uso de objetos para mapear teclas e sons, facilitando a manutenção e escalabilidade do código.

### Manipulação do DOM

Seleção de elementos, manipulação de eventos e atualização dinâmica da interface.

### Tratamento de Eventos

Trabalho com eventos de clique e teclado para melhorar a interação com o usuário.
